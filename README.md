# 개발환경
| Language | JAVA (JDK 1.8) |
| --- | --- |
| Infra | AWS EC2(Ubuntu 20.04), RDS, S3 |
| Framework | Spring Framework 2.7.1, Hibernate |
| DB | MySQL |
| Library, API | JPA, Naver Cloud Platform, JWT |

| Collaboration | Slack, Swagger, Figma, Discord, Jira |
